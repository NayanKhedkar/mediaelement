Welcome to MediaElement.js.

MediaElement is a wrapper that mimics the native HTML5 MediaElement syntax (get/set) as a renderer that can handle media from HTML5, YouTube, Vimeo, Soundcloud, Flash, HLS, and other libraries.

MediaElementPlayer is built off of MediaElement and creates a fully featured player on top of the unified syntax from MediaElement.

MediaElement.js supports IE9+, MS Edge, Chrome, Firefox, Safari, iOS 8+ and Android 4.0+

MediaElement.js has renderers to play the following media formats.

What's New on 3.0 version

  • Introduction of Renderers, pluggable code that allows the introduction of new media formats in an easier way.
  • Ability to play Facebook, SoundCloud, M(PEG)-DASH using dash.js for native support and dash.as for Flash fallback.
  • Addition of native HLS using hls.js library.
  • Updated player for Vimeo by removing the use of Froogaloop and integrating the new Player API.
  • Support for FLV natively using flv.js library.
  • ... and more.
